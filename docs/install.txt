LiveRecordingReview Installation Instructions:

1. 3rd Party library additional install requirements:

WEB-INF/lib/
	ant-antlr.jar [1]
	antlr.jar [1]
	asm-attrs.jar [1]
	asm.jar [1]
	cglib.jar [1]
	commons-collections.jar [1]
	commons-dbcp.jar [2]
	commons-logging.jar [1]
	commons-pool.jar [2]
	dom4j.jar [1]
	hibernate3.jar [1]
	jstl.jar [3]
	jta.jar [1]
	log4j.jar [1]
	mysql-connector-java-3.1.13-bin.jar [4]
	spring.jar [2]
	standard.jar [3]

[1] Hibernate 3.2 distribution (http://www.hibernate.org/)
[2] Spring 2.0 distribution (http://www.springframework.org/)
[3] Jakarta Standard Taglibs 1.1.2 distribution (http://jakarta.apache.org/taglibs/)
[4] MySQL Connector/J JDBC distribution (http://dev.mysql.com/downloads/connector/j/3.1.html)

2. Create database of your choice and run docs/init-database.sql script
3. Edit WEB-INF/liverecordingreview-data.xml to use dataSource url/username/password or your choice and hibernate dialect (if not using MySQL)
4. Run Ant "install" target to install to Tomcat 5.5.x +
